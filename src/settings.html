Copilot Agent Traces Settings

Connect to Application Insights using Azure AD authentication. No secrets are stored.
Azure Portal → Application Insights → API Access → Application ID
Azure Portal → App registrations → your app → Overview → Application (client) ID
Azure Portal → App registrations → your app → Overview → Directory (tenant) ID

Setup Instructions

  1. Go to Azure Portal → App registrations → New registration.
  2. Name: Copilot Trace Viewer, Single tenant, Redirect URI: SPA → https://andrewscheuer.gallerycdn.vsassets.io
  3. Go to API permissions → Add → APIs my organization uses → search Application Insights API → Delegated → Data.Read → Add.
  4. Click Grant admin consent.
  5. Copy Application (client) ID and Directory (tenant) ID from Overview and paste above.
  6. Go to your Application Insights resource → API Access → copy the Application ID and paste above.
  7. Click Save Settings. The Agent Traces tab will authenticate automatically via Azure AD.
