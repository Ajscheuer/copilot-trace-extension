Copilot Agent Traces Settings

Connect to Application Insights to view Copilot agent traces on work items.
Found in Azure Portal → Application Insights → API Access → Application ID
Create one in API Access → Create API Key → select "Read telemetry"

Setup Instructions

  1. Open the Azure Portal and navigate to your Application Insights resource.
  2. Go to API Access (under Configure).
  3. Copy the Application ID and paste it above.
  4. Click Create API Key, select "Read telemetry", and paste the key above.
  5. Click Save Settings. The Agent Traces tab will appear on all work items.
  6. Ensure developers have github.copilot.chat.otel.enabled: true in VS Code.
  7. Tag traces with work item IDs via branch naming (e.g., feature/1234-description).
